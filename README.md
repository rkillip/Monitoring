# LBSU Golf Recruit Overlay Tool

A recruiting tool for Long Beach State Men's Golf that:
- Scrapes tournament leaderboard data from Clippd Scoreboard
- Maintains a watchlist of transfer portal prospects
- Overlays a recruit's SG data onto your schedule to project finish position, points, and team placement

---

## Setup

### 1. Install dependencies
```bash
pip install requests beautifulsoup4 pandas
```

### 2. Find your tournament IDs
Tournament IDs are in the Clippd Scoreboard URL:
```
https://scoreboard.clippd.com/tournaments/238883/scoring/player
                                           ^^^^^^
                                        This is the ID
```

Go to each of your season's tournament pages and copy the ID number.

### 3. Add your tournament IDs to `scraper.py`
In `scraper.py`, find the `LBSU_TOURNAMENTS` dictionary and add all your events:
```python
LBSU_TOURNAMENTS = {
    238883: "Thunderbird Collegiate",
    123456: "Ram Masters",
    123457: "Tucker Intercollegiate",
    # ... etc
}
```

### 4. Run the scraper
```bash
python scraper.py
```
This creates `data/tournament_data.csv` with full field leaderboards for all your events.

### 5. Run the watchlist overlay
```bash
python watchlist.py
```
This will:
- Create `data/watchlist.json` seeded with your current prospects (Jorge, Rasmus, Alvaro, Drew)
- Run the schedule overlay for any recruit who has SG data entered
- Show projected finish, points, and team placement for each tournament

---

## Weekly Update Workflow (Every Wednesday/Thursday)

1. Check if any new tournaments have posted on Clippd
2. Add new tournament IDs to `scraper.py`
3. Run `python scraper.py` to pull fresh data
4. Run `python watchlist.py` to update projections

---

## Adding a New Recruit to the Watchlist

Edit `watchlist.py` and add a new `Recruit` object, then call `add_recruit()`:

```python
from watchlist import Recruit, RecruitSGEntry, add_recruit

new_recruit = Recruit(
    name="Player Name",
    school="Their School",
    country="USA",
    points_avg=45.0,
    scoring_avg=71.2,
    ranking="#250",
    notes="Your scouting notes here",
    sg_data=[
        RecruitSGEntry("Ram Masters", -0.5),      # Their SG that week
        RecruitSGEntry("Thunderbird", 1.2),
        # Add one entry per tournament on your schedule
        # Use the tournament name exactly as listed in LBSU_TOURNAMENTS
    ]
)
add_recruit(new_recruit)
```

---

## Adding SG Data for a Recruit

Go to the player's profile on Clippd Scoreboard, find their SG for each event,
and add it to their `sg_data` list. The tournament name must match exactly what's
in your `LBSU_TOURNAMENTS` dictionary.
